# Dotfiles managed by chezmoi

### Download .laptop.local

```bash
curl --remote-name https://raw.githubusercontent.com/thtran082/dotfiles/main/dot_laptop.local
mv dot_laptop.local .laptop.local
chmod +x .laptop.local
zsh .laptop.local
```
